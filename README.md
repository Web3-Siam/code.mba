# code.mba

### Installation
With [npm](https://www.npmjs.com/):
`npm install`

With [yarn](https://yarnpkg.com/):
`yarn install`

### Local Server
With [npm](https://www.npmjs.com/):
`npm run dev`

With [yarn](https://yarnpkg.com/):
`yarn dev`

### Build
With [npm](https://www.npmjs.com/):
`npm run build`

With [yarn](https://yarnpkg.com/):
`yarn build`
